# intdiv

```php
	$x = 10;
	$y = 3;
	intdiv($x, $y); // 3
	( $x - ( $x % $y ) ) / $y; // 3 
```

<Notes>
	Returns the integer 
	Helpful for something like pagination
</Notes>
